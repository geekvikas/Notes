# Get Summary Info of Multiple Files and Folders

## Problem
* Want to Get Total Size of Multiple Selected Files

## Solution
* Select the Files
* Keep Pressing "Ctrl" and Click(Both Right and Left Click Work) to Open the Menu
* Select "Get Summary Info" Item
