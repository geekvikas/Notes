# Set q-btn Width and Text Size

## Problem
Need to set the width of the q-btn and text size.

## Example
```vue
  <q-card-actions align="center">
    <q-btn 
      style="width: 60%;"
      size="4vh"
      rounded
      color="teal"
      label="登录"
      type="submit"
      @click="onLogin"
      />
  </q-card-actions>
```
