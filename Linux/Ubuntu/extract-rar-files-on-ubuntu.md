# Extract Rar Files on Ubuntu

## Solution
* Install `rar` and `unrar` on Ubuntu

  ```bash
  sudo apt install rar unrar
  ```

* Extract Rar Files

  ```bash
  unrar e my.rar /path/to/dir
  ```
