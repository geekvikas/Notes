# Can not Input Correct Password to Login due to Incorrect Input Method Selected

## Problem
* Input the password as usual but it reported wrong password and failed to login
* Num pad area does not work

## Root Cause
* On the login UI, Clicked the right-bottom "Input Method" icon and changed the input method(if user has 2 or more input methods)
* "DEU" indicated that "Deutschland" input method was selected

## Solution
* Click the "Input Methed" icon again to change the input method back(e.g. Chinese Pinyin)

# 由于选择了错误的输入法导致不能输入正确的密码进行登录

## 问题
* 正常输入往常的密码，提示密码错误不能登录
* 小键盘区没有响应

## 原因
* 在登录页面，不小心点击了右下角的输入法图标，改变了输入法（如果用户有2个或以上的输入法）
* "DEU"表示选择了德国的输入法

## 解决方法
* 再次点击输入法图标，将正确的输入法改回来（e.g. 拼音输入法）
