# Excel 中删除身份证号前的单引号

## 问题
* 在复制粘贴身份证的时候，单元格格式没有预先设置为“文本”，默认为常规
* 导致粘贴后，单元格内的身份证前，被自动添加了`'`单引号

  ```
  '310104201008010000
  ```
 
* 需要批量删除身份证号前的这些单引号

## 解决
* 使用公式，提取右侧的 18 位字符

  ```
  =RIGHT(A1,18)
  ```
