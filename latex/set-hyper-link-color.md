# Set Hyper Link Color

## Solution
Use `hypersetup` command.

## Example
```latex
\hypersetup{
    colorlinks=true,
    linkcolor=blue,
    filecolor=magenta,
    urlcolor=cyan,
    pdftitle={My PDF Title},
    pdfpagemode=FullScreen,
}
```
