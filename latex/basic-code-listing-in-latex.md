# Basic Code Listing in Latex

## Solution
Use the verbatim environment.

## Example
```
\section{Installation}

To install tex2pdf package, run the command:

\begin{verbatim}
go get github.com/northbright/tex2pdf@latest
\end{verbatim}
```
