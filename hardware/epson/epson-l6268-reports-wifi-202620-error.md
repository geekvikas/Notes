# Epson L6268 Reports Wi-Fi `202620` Error

## Problem
* EPSON L6268 Reports "Wi-Fi Error"(code: `202620`)

## Root Cause
It's probable that the Wi-Fi Module is loosen.

## Solution
* Reconnect the WiFi Module Cable or Contact EPSON Service
