# 因为交换机级连层数过多，导致有一些海康威视摄像头不能 ping 通

## 问题
* 一些海康威视摄像头可以 ping 通，另外一些不能

## 原因
* 不能 ping 通的摄像头，连接的交换机级连层数过多
* 拓扑图

```
        ┌─────────────┐
        │ Core Switch │
        └──┬──────────┘
           │
        ┌──▼────────────┐
        │ Access Switch │
        └───┬───────────┘
            │
        ┌───▼───────────┐
        │ Access Switch │
        └──┬─────────┬──┘
           │         │
           │         │
  ┌────────▼──┐   ┌──▼─────────────┐
  │  Camera 1 │   │ Layer 2 Switch │
  └───────────┘   └─────┬──────────┘
                        │
                      ┌─▼────────┐
                      │ Camera 2 │
                      └──────────┘
```

  * Camera 1 可以 ping 成功
  * Camera 2 ping 失败，因为级连层数过多，最后还级连 1 个 2 层交换机
