# 隐藏 / 显示连接到萤石云的海康威视的 NVR 中的通道

## 问题
* 海康威视 NVR，扫描设备上的平台接入萤石云的二维码，连接到萤石云
* 32路通道，只使用了部分通道
* 萤石云显示了全部通道，包括未使用的部分，点击画面显示“加载失败”
* 需要将未使用的通道隐藏

## 解决
* 在萤石云 App > 选择设备 > 点击底部设备名称右侧的一个“:" > 进入设置

```

│                                                                     32 Cameras │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
│                                                                                │
├────────────────────────────────────────────────────────────────────────────┬─┬─┤
│                                                                            ├─┤ │
│  NVR Device Name(e.g. My NVR)                                              │:│ │
│                                                                            ├─┤ │
└────────────────────────────────────────────────────────────────────────────┘▲└─┘
                                                                              │
                                                                              │
                                                                              │
                                                                              │
                                                             ┌────────────────┴──┐
                                                             │  Click This Colon │
                                                             │                   │
                                                             │ to Enter Settings │
                                                             └───────────────────┘
```

* 本地设备未添加的通道，会显示“未关联”，选择这些摄像头
* 选择进入相应的通道 > 高级设置 > 隐藏通道
